# NGO Fund Tracker

## Problem
NGOs often struggle to maintain transparent financial records for donations and expenses.

## Solution
NGO Fund Tracker provides a centralized platform to manage donors, donations, projects, and expenses with real-time tracking.

## Tech Stack
- FastAPI (Backend)
- Streamlit (Frontend)
- SQLite (Database)
- Python

## Features
- Donor Management
- Donation Tracking
- Project Budget Management
- Expense Tracking
- Dashboard Analytics

## How to Run

1. Start Backend
uvicorn backend.main:app --reload

2. Start Frontend
streamlit run frontend/app.py
